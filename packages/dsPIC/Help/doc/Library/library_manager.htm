Library Manager

Library Manager enables simple handling libraries being used in a project. Library Manager window lists all libraries (extension .mcl) which are instantly stored in the compiler Uses folder.
The desirable library is added to the project by selecting check box next to the library name (only the selected libraries will be linked).

In order to have all library functions accessible, simply press the button Check All and all libraries will be selected.
In case none library is needed in a project, press the button Clear All and all libraries will be cleared from the project.

Library Categorization

Libraries listed in the Library Manager are grouped in three general categories (also presented on the picture below) :

  • System Libraries - Default libraries available upon compiler installation (cannot be uninstalled),
  • mikroSDK Libraries - mikroSDK related libraries, usually created for Click boards,
  • Additional Libraries - Various other libraries.

Handling Library Routines

To add the desired routine name to your current source file, simply drag and drop the routine from the library node to Code Editor or press Insert when routine is selected in the Library Manager.

The routine prototype will pop-up when mouse pointer is positioned above the desired routine.
